# Procédure d'extraction des styles Figma via API REST

## Token d'accès Figma

Créez un token personnel via les paramètres Figma et stockez-le localement
dans une variable d'environnement, par exemple :

```bash
export FIGMA_ACCESS_TOKEN="votre_token_figma"
```

⚠️ **Important**: Ne stockez jamais votre token directement dans le dépôt.
